
### FFmpeg

비디오 처리를 위해 FFmpeg 바이너리가 필요

1. **FFmpeg 다운로드:**

    - [FFmpeg 공식 웹사이트](https://ffmpeg.org/download.html)에서 운영 체제에 맞는 바이너리를 다운로드

2. **FFmpeg를 프로젝트 디렉토리에 추가:**

    - 다운로드한 FFmpeg 바이너리를 아래와 같은 디렉토리 구조로 프로젝트에 추가

    ```
    루트폴더/
    └── ffmpeg/
        └── ffmpeg (또는 Windows의 경우 ffmpeg.exe)
    ```
